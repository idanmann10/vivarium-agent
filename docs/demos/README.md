# Demo Recordings

- `local-e2e.cast` is an asciinema v2 recording generated by `bun run record:local-e2e-demo`.
  It demonstrates local `init`, a deterministic `run`, second-install `world transmission-smoke`,
  and the Node-side `verify:sqlite-stack` migration verifier.

The recording is a local checkpoint artifact, not proof that live v1 is complete.
Live provider calls, GitHub writes, final remotes, and real cross-install pulls
are tracked separately in `docs/guides/live-readiness.md` and the v1 completion
audit.
