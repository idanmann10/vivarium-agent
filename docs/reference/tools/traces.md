# Traces Tool

The `traces` self-tool reads teaching traces from the world and writes local trace candidates from instructive runs.

- `search(topic, domain?)` searches subscribed world content for traces.
- `read(id, domain?)` returns one matching trace result.
- `author(runId, annotations, domain?)` creates a trace candidate with annotated steps for later publishing.
