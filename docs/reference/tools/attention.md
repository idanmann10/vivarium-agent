# Attention Tool

Focuses or defocuses skills, tools, and episodes in the active working set.

- `focus(request)` narrows active caps for skills, tools, tokens, or episodes.
- `defocus()` restores default attention caps.
- `status()` reports whether focus is active, the current caps, and current usage fields.
